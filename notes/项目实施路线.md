# 项目实施路线

## 阶段 0：资料确认

目标：消除方向性不确定。

- 确认最终运行系统：Win10 或 Win7。
- 确认 Zebra USB 底层连接类型。
- 获取串口私有协议文档。
- 确认 SECS/GEM 角色、连接参数和真实报文。
- 确认标签模板和验收样张。

## 阶段 1：打印最小闭环

目标：从业务输入到真实打印。

- 设计 ZPL 模板。
- 将打印内容字段映射为一维码/文本。
- 封装打印接口：
  - 打印成功
  - 打印机离线
  - 缺纸
  - 内容为空
  - 连接失败
- 输出可重复测试的小工具。

## 阶段 2：SECS/GEM-HSMS 最小闭环

目标：Host 下发打印，EQ 返回执行结果。

- 建立 HSMS 连接。
- 实现 `S1F1/S1F2` 心跳。
- 实现 `S8F3/S8F4` 打印指令。
- 打印成功/失败映射到协议结果码。

## 阶段 3：串口设备接入

目标：把电子货架格口/RFID 数据接入系统。

- 串口连接、重连、日志。
- 查询格口状态。
- 读取/设置 RFID。
- 错误码与异常处理。

## 阶段 4：完整协议

目标：补齐协议文档中的业务能力。

- `S5F11/S5F12` 查询格口。
- `S6F21` 格口状态主动上报。
- `S10F11/S10F12` 设置 RFID。
- `S6F11` 通用事件上报。
- 报警 ALID 映射。

## 阶段 5：交付

目标：现场可运行、可排障。

- 配置文件：串口、打印机、HSMS、货架编号。
- 本地日志：通讯日志、打印日志、异常日志。
- 安装/部署说明。
- 验收用例。
